## What is Galaxies UI?

Galaxies UI is an extend for Shadcn/ui component library that provides a set of components for building web applications. It is built on top of <Link target="_blank" href="https://ui.shadcn.com/" >Shadcn/ui</Link>.

## Capabilities

Galaxies UI is a set of components that are built on top of Shadcn-ui . It extends the component that are provided by shadcn and use the one there to build some common component for the most web apps . It is built with the following goals in mind:

- **Customizable**: Galaxies UI is designed to be highly customizable. It provides a set of components that can be easily customized to fit your needs.
- **Easy to use**: Galaxies UI is designed to be easy to use. It provides a set of components that are easy to use and can be easily integrated into your web application.
- **No need for installation**: Galaxies UI does not require any installation. just Copy&Paste the code and install the shadcn/ui component that it uses.

## Getting Started

To get started with Galaxies UI, you need to have a basic understanding of ReactJs/NextJs and Tailwinds .You also need to have a basic understanding of Shadcn ui. If you are new to Shadcn-ui, you can check out the <Link target="_blank" href="https://ui.shadcn.com/" >Shadcn/ui</Link>.

## Installation

To install Galaxies UI, you need to configure the shadcn/ui component that it uses. You can do this by following the instructions in the <Link target="_blank" href="https://ui.shadcn.com/" >Shadcn/ui</Link>.

## Usage

To use Galaxies UI, you need to Copy&Paste the components that you want to use from the **Galaxies UI** . You can then use these components in your web application.

Example:

```tsx
import React from "react";
import { BreadCrumb } from "@/component/extension/breadcrumb";

export const BreadCrumbTest = () => {
  return (
    <BreadCrumb variant="ghost">
      <BreadCrumbItem>Home</BreadCrumbItem>
      <BreadCrumbSeparator />
      <BreadCrumbItem>Settings</BreadCrumbItem>
      <BreadCrumbSeparator />
      <BreadCrumbItem>Info</BreadCrumbItem>
    </BreadCrumb>
  );
};
```
